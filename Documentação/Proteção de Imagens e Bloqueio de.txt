## Por que você perde sua barra de rolagem personalizada?

O Locomotive Scroll, por padrão, assume o controle da rolagem da página, o que significa que ele "esconde" a barra de rolagem nativa do navegador e utiliza sua própria lógica para criar o efeito de rolagem suave. É por isso que sua barra de rolagem personalizada desaparece.

## Como resolver esse problema?

A solução é basicamente "dizer" ao Locomotive Scroll para não interferir na sua barra de rolagem personalizada. Para isso, você precisa configurar o Locomotive Scroll para funcionar em conjunto com a sua barra de rolagem, em vez de substituí-la.

## Passo a passo

1. **Estrutura do HTML:** Certifique-se de que sua barra de rolagem personalizada (`.c-scrollbar`) e o conteúdo da página estejam dentro de um contêiner com uma classe específica (ex: `.smooth-scroll`).

```html
<div class="smooth-scroll" data-scroll-container>
  <div class="c-scrollbar">
    <div class="c-scrollbar_thumb"></div>
  </div>

  <section id="hero">...</section>
  <section id="albums">...</section>
  </div>
```

2. **Inicialização do Locomotive Scroll:** Ao inicializar o Locomotive Scroll, você precisa especificar o contêiner que terá a rolagem controlada por ele (o mesmo contêiner que envolve sua barra de rolagem e o conteúdo).

```javascript
const locoScroll = new LocomotiveScroll({
  el: document.querySelector('.smooth-scroll'), // Seu container
  smooth: true // Ativa a rolagem suave
});
```

3. **CSS:** Ajuste o CSS para que sua barra de rolagem personalizada funcione corretamente dentro do contêiner `.smooth-scroll`.

```css
.smooth-scroll {
  /* Garante que o conteúdo tenha altura suficiente para a rolagem */
  height: 100%;
  overflow-y: auto; /* Esconde a barra de rolagem nativa */
}

.c-scrollbar {
  /* Posicione sua barra de rolagem personalizada */
  position: absolute;
  top: 0;
  right: 20px; /* Ajuste a posição conforme necessário */
  /* ... (resto do seu CSS para a barra de rolagem) */
}
```

4. **JavaScript:**
    * **Sincronização da barra de rolagem:** Você precisará adicionar um pouco de JavaScript para sincronizar a posição da sua barra de rolagem personalizada com a rolagem do conteúdo controlada pelo Locomotive Scroll.

```javascript
locoScroll.on('scroll', (event) => {
  const scrollHeight = event.scroll.height;
  const scrollTop = event.scroll.y;
  const thumbHeight = (window.innerHeight / scrollHeight) * 100;
  const thumbPosition = (scrollTop / scrollHeight) * 100;

  scrollThumb.style.height = `${thumbHeight}%`;
  scrollThumb.style.transform = `translateY(${thumbPosition}%)`;
});
```

## Explicação do código

* O seletor `.smooth-scroll` define o contêiner que terá a rolagem controlada pelo Locomotive Scroll.
* O atributo `data-scroll-container` no HTML é necessário para o Locomotive Scroll identificar o contêiner de rolagem.
* A propriedade `smooth: true` na inicialização do Locomotive Scroll ativa a rolagem suave.
* O evento `scroll` do Locomotive Scroll é usado para obter a posição atual da rolagem e atualizar a posição da sua barra de rolagem personalizada.

## Observações importantes

* **Conflitos de CSS:** Certifique-se de que não haja conflitos de CSS entre o Locomotive Scroll e o seu estilo personalizado.
* **Versão do Locomotive Scroll:** Este código é baseado na versão 1 do Locomotive Scroll. Se você estiver usando uma versão diferente, pode haver algumas diferenças na API.
